Part 1

This app is can be used to analyze the crab dataset originally contained in the R library MASS.

This data frame contains the following columns:

  • "sp", which means species--"B" or "O" for blue or orange.
  • "sex", as it says, the gender, Male or Female
  • "FL", which means frontal lobe size (mm)
  • "RW", which means rear width (mm)
  • "CL", which means carapace length (mm)
  • "CW", which means carapace width (mm)
  • "BD", which means body depth (mm)

    • The first tab shows the data, allowing the user to filter according to sex or species, being able to watch only one subgroup or both.

    The second tab shows an interactive scatter plot, where the user can decide which variables to compare. It also allows the user to specify the discriminating color, sex or species. This last part is useful for the next problem, if we wanted to classify a crab based on a couple measures, with this plot we know that for either sex or species we will have two clearly different populations.
